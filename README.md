# Jogo da Forca em C# #

### Este é um jogo da forca simples desenvolvido em C# para o console. O jogo escolhe uma palavra secreta e o jogador tenta adivinhar a palavra, letra por letra. Se o jogador errar 6 vezes, ele perde o jogo. ###

### Como Jogar ###

#### Regras do Jogo ####

Executar o Jogo: Compile e execute o código C# em um ambiente de desenvolvimento .NET ou usando o compilador csc no terminal.

Adivinhar a Palavra: O jogo exibirá a palavra secreta com letras ocultas por underscores (_).

Inserir Letras: O jogador digita uma letra e pressiona Enter.

Verificar Letras:
Se a letra estiver na palavra secreta, ela será revelada nas posições corretas.
Se a letra não estiver na palavra, o número de erros aumentará.

Vencer ou Perder:
O jogador vence se adivinhar todas as letras da palavra secreta antes de atingir 6 erros.
O jogador perde se atingir 6 erros antes de adivinhar a palavra secreta.

Jogar Novamente: O jogo continuará rodando até que o usuário feche o programa.
